# AaserZypher Next.js Starter Template

Custom-built minimal Next.js starter built for https://aaserzypher.dev.

## Features

- Pages Router (Next JS 15)
- Reusable components
- TailwindCSS
- Typescript

## Getting Started

1. Clone the repository

```bash
npm install
npm run dev
```

2. Navigate to http://localhost:3000
